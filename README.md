# mastodon-bber

A Chrome extension for quickly posting to Mastodon with support for multiple accounts, content warnings, and convenient interaction with your timeline.

## Overview

mastodon-bber is a lightweight Chrome extension that lets you post to Mastodon directly from your browser. It focuses on quick posting, simple account switching, and tools that help you interact with your own posts more efficiently.

## Features

* **Quick Tooting**
  Post to Mastodon directly from the extension popup.

* **Multiple Account Support**
  Add and switch between different Mastodon accounts or instances.

* **Content Warning (CW)**
  Add a Content Warning to your toot before posting.

* **Reply with Your Latest Extension Toot**
  Easily reply to a toot using the most recent toot you posted through the extension.

* **Featured Hashtag Reader**
  View posts from a featured hashtag and quickly choose one to toot or interact with.

## Installation

### Install from Source

1. Clone the repository:

```bash
git clone https://github.com/yourusername/mastodon-bber.git
```

2. Open Chrome and go to:

```
chrome://extensions
```

3. Enable **Developer Mode**.

4. Click **Load unpacked**.

5. Select the `mastodon-bber` project folder.

The extension will now appear in your browser toolbar.
